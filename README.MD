## Описание проекта
Данное консольное приложение разработано в рамках выполнения 3-й модульной работы по дисциплине «ООП». Проект демонстрирует навыки работы с наследованием классов, обработкой `null`-значений, использованием кортежей и механизмами Pattern Matching (сопоставление с образцом) в C#.

## Основной функционал
Программа разделена на логические модули для удобства поддержки кода:

* **Модели (Models):**
    * `Book`, `Textbook`, `PoetryBook`: классы для иерархии книг.
    * `Appliance`: класс для описания бытовой техники.
* **Логика (Logic):**
    * `BookUtils`: метод генерации описания книги в зависимости от ее типа.
    * `ApplianceUtils`: логика расчета скидок на технику (ноутбуки, планшеты).
    * `WeatherUtils`: обработка кортежей с данными о погоде для принятия решений.
    * `ListUtils`: анализ списков с помощью Pattern Matching (проверка структуры списков).

## Как запустить
1. Убедитесь, что установлен .NET SDK.
2. Склонируйте репозиторий.
3. Выполните команду `dotnet run` в терминале из корневой папки проекта.
